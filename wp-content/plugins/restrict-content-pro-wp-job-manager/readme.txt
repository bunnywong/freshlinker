Limit job submissions through WP Job Manager to paid subscribers in Restrict Content Pro

== Description ==

Limit job submission and / or applications for [WP Job Manger](https://github.com/mikejolley/WP-Job-Manager) to paid subscribers in [Restrict Content Pro](http://pippinsplugins.com/restrict-content-pro-premium-content-plugin/) This extension does NOT function without WP Jobs Manager AND Restrict Content Pro installed.

This extension has no options and will automatically require posters to have an active subscription in order to post jobs.

Learn more about Restrict Content Pro: http://pippinsplugins.com/restrict-content-pro-premium-content-plugin/

2. Activate
